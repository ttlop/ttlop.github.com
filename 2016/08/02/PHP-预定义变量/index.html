PHP 预定义变量

  • $_SERVER[“DOCUMENT_ROOT”]
    根目录，在 apache 的配置文件里面定义：比如 httpd.conf 中的 DocumentRoot “/var/www/html”

  • $_SERVER[‘REMOTE_ADDR’]
    你的客户端跟你的服务器“握手”时候的 IP，如果使用了代理，REMOTE_ADDR 将显示代理服务器的IP

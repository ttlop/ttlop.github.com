PHP feof() 函数

定义和用法

fe() 函数检测是否已经达到文件末尾(eof)。
如果文件指针到了 EOF 或者出错时则返回 TRUE，否则返回一个错误（包括 socket 超时），其它情况则返回 FALSE。

语法

参数 描述
file 必须。规定要检查的打开文件。

说明：file 参数是一个文件指针。这个指针必须有效，并且必须指向一个由 fopen() 或 fsockopen() 成功打开（但还没有被 fclose()关闭）的文件。

提示和注释

提示：feop() 函数对遍历程度未知的数据很有用。

注意：如果服务器没有关闭由 fsockopen() 所打开的连接，feof() 会一直等待直到超时而返回 TRUE。默认的超时限制是 60 秒，可以使用 stream_set_timeout() 来改变这个值。

注意：如果传递的文件指针无效可能会陷入无线循环中，因为 EOF 不会返回 TRUE。

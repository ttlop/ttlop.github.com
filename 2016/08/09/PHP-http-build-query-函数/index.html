PHP http_build_query 函数

http_build_query - 生成 URL-encode 之后的请求字符串。简单来说，http_build_query() 就是将一个数组换成 url 问号后面的参数字符串，并且会自动进行 urlencode 处理。

说明

使用给出的关联（或下标）数组生成一个经过 URL-encode 的请求字符串。query_data 可以是数组或包含属性的对象，可以是简单的一维结构，也可以是由数组组成的数组。

Zabbix Template Windows 2008 R2 IIS Servers

DownUrl

Usage

1. Import

Configuration -> Templates — Import

2. Template Name

Template Windows 2008 R2 IIS Servers

3. Items

参考：http://doc.51windows.net/iismmc/?url=/iismmc/htm/ref_we_counters.htm

Name KEY Comment
Windows 2008 R2 IIS Web Service Cache Current Metadata Cached perf_counter[“\Web Service Cache\Current Metadata Cached”,300] 用户模式缓存中的元数据信息块的当前数量
Windows 2008 R2 IIS Web Service Cache Current Files Cached perf_counter[“\Web Service Cache\Current Files Cached”,300] 自 WWW 服务启动后，在用户模式文件缓存中查找成功的总数
Windows 2008 R2 IIS Web Service Cache Current URIs Cached perf_counter[“\Web Service Cache\Current URIs Cached”,300] 自 WWW 服务启动后，在用户模式 URI 缓存中成功查找的总数
Windows 2008 R2 IIS Web Service Cache File Cache Hits % perf_counter[“\Web Service Cache\File Cache Hits %”,300] 自 WWW 服务启动后，用户模式文件缓存命中与缓存请求总数的比
Windows 2008 R2 IIS Web Service Cache URI Cache Hits % perf_counter[“\Web Service Cache\URI Cache Hits %”,300] 自 WWW 服务启动后，URI 缓存命中与缓存总请求数的比
Windows 2008 R2 IIS Web Service Cache Metadata Cache Hits perf_counter[“\Web Service Cache\Metadata Cache Hits”,300] 自 WWW 服务启动后，在用户模式元数据缓存中成功查找的总数
Windows 2008 R2 IIS Web Service(_Total) Post Requests/sec perf_counter[“\Web Service(_Total)\Post Requests/sec”,300] 每秒使用 POST 方法对 WWW 服务进行的 HTTP 请求数
Windows 2008 R2 IIS Web Service(_Total) Get Requests/sec perf_counter[“\Web Service(_Total)\Get Requests/sec”,300] 每秒使用 GET 方法对 WWW 服务进行的HTTP 请求数
Windows 2008 R2 IIS Service IIS World Side Web Publishing service_state[W3SVC] 当前是否有开启W3SVC服务
Windows 2008 R2 IIS Service IIS Admin service_state[IISAdmin] 当前是否有开启IISAdmin服务
Windows 2008 R2 IIS Web Service(_Total) Bytes Received/sec perf_counter[“\Web Service(_Total)\Bytes Received/sec”,300] WWW 服务每秒接收的数据字节数
Windows 2008 R2 IIS Web Service(_Total) Bytes Sent/sec perf_counter[“\Web Service(_Total)\Bytes Sent/sec”,300] WWW 服务每秒发送的数据字节数
Windows 2008 R2 IIS Web Service(_Total) Current Connections perf_counter[“\Web Service(_Total)\Current Connections”,300] 当前与 WWW 服务建立连接的数量
Windows 2008 R2 IIS Web Service(_Total) Bytes Total/sec perf_counter[“\Web Service(_Total)\Bytes Total/sec”,300] bytes sent/sec 和 bytes received/sec 的总和
Windows 2008 R2 IIS Port 80HTTP net.tcp.listen[80] 当前是否有侦听TCP80端口

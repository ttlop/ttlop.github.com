Zabbix 配置第三方邮件服务器实现邮件预警

Media Types Configuration

我们使用联通的 wo 邮箱发送报警邮件到 qq 邮箱

Item Value Comment
Name Email 媒介名称，自定义设置
Type Email 本篇文章我们以 Email 为例
SMTP server smtp.wo.cn smtp服务器地址，我们使用联通的
SMTP server port 25 smtp连接端口
SMTP helo smtp helo值，通常是顶级域名，这个地方需要输入 smtp.wo.cn
Connection security None 是否加密，这里选择 None
Authentication Normal password 认证方式，这里选择 Normal password
Username 18675850419 输入 wo 邮箱账号，必须要加后缀部分
Password xxxx 输入对应的 wo 邮箱密码
Enabled 勾选 勾选启用

Users Configuration

  1. 这里我们配置系统默认的 Admin 账户，在 Media 页添加上一步新增的介质

  2. 需要配置 Send to，其后填写接收报警信息的邮件地址，此处我填写我的 411383444@qq.com 邮箱

Actions Configuration

  1. 这里我们配置系统默认的 Report problems to Zabbix administors action

  2. 需要勾选 Action 页的 Enabled，如需要，也请一并勾选 Recovery message

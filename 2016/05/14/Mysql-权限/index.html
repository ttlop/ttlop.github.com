Mysql 权限

show privileges 显示服务器支持的权限

权限级别 权限 权限说明
表 alter To alter the table
存储过程 alter routine To alter or drop stored functions/procedures
数据库、表或索引 create To create new databases and tables
数据库 create routine To use CREATE FUNCTION/PROCEDURE
数据库 create temporary tables To use CREATE TEMPORARY TABLE
服务器管理 create user To create new users
表 create view To create new views
表 delete To delete existing rows
数据库或表 drop To drop databases, tables, and views
服务器管理 event To create, alter, drop and execute events
存储过程 execute To execute stored routines
服务器主机上的文件访问 file To read and write files on the server
数据库、表或存储过程 grant option To give to other users those privileges you possess
表 index To create or drop indexes
表 insert To insert data into tables
数据库 lock tables To use LOCK TABLES (together with SELECT privilege)
服务器管理 process To view the plain text of currently executing queries
服务器管理 proxy To make proxy user possible
数据库、表 references To have references on tables
服务器管理 reload To reload or refresh tables, logs and privileges，例如flush status
服务器管理 replication client To ask where the slave or master servers are
服务器管理 replication slave To read binary log events from the master
表 select To retrieve rows from table
服务器管理 show database To see all databases with SHOW DATABASES
表 show view To see views with SHOW CREATE VIEW
服务器管理 shutdown To shut down the server
服务器管理 super To use KILL thread, SET GLOBAL, CHANGE MASTER, etc.
表 trigger To use triggers
服务器管理 create tablespace To create/alter/drop tablespaces
表 update To update existing rows
服务器管理 usage No privileges - allow connect only

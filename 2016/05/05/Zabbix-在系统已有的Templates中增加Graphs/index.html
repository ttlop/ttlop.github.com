Zabbix 在系统已有的Templates中增加Graphs

这里讲述如何在zabbix系统自带的templates中增加graphs项。
zabbix版本：zabbix 3.0.1
templates名：Template OS Linux

1.选择点击导航条上的Configuration-Tempplates

2.选择Template OS Linux，点击后面的Graphs

3.点击右上角Create graph

4.在Name中输入我们创建的这个Graph的名字，在items中选择我们创建的这个Graph上需要显示的数据项

5.点击Add完成创建

6.可以通过Configuration-Hosts中添加的主机的Graphs查看我们创建的graph是否成功
注：前提是我们在主机中应用了前面所说的Template OS Linux

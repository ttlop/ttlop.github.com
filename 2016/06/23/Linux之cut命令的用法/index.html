Linux之cut命令的用法

cut - 输出文件行的选定部分到标准输出。
如果不指定 File 参数，cut 命令将读取标准输入。必须指定 -b、-c 或 -f 标志之一。

主要参数

Option Content
-b 以字节为单位进行分隔。这些字节位置将忽略多字节字符边界，除非也指定了 -n 标志
-c 以字符为单位进行分隔
-d 自定义分隔符，默认为制表符
-f 与 -d 一起使用，指定显示哪个区域
-n 取消分隔多字节支付。仅和 -b 一起使用。如果字符的最后一个字节落在由 -b 标志的 list 参数指示的范围之内，该字符将被写出；否则，该字符将被排除

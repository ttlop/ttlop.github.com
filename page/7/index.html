PHP trim() 函数

定义和用法

trim() 函数移除字符串两侧的空白字符或其它预定义字符。

语法

参数 描述
string 必须，规定要检查的字符串
charlist 可选，规定从字符串中删除哪些字符。如果被省略，则移除以下字符
  • “\0” - NULL
  • “\t” - 制表符
  • “\n” - 换行符
  • “\x0B” - 垂直制表符
  • “\r” - 回车
  • “” - 空格

技术细节

返回值：返回被修改的字符串
PHP版本：4+
更新日志：在 PHP 4.1 中，新增了 charlist 参数

PHP strpos() 函数

定义和用法

strpos() 函数查找字符串在另一个字符串中第一次出现的位置，大小写敏感。

语法

参数 描述
string 必须，规定要搜索的字符串
find 必须，规定要查找的字符串
start 可选，规定在何时开始搜索

技术细节

返回值：返回字符串在另一个字符串中第一次出现的位置，如果没有找到字符串则返回 FALSE。字符串位置从 0 开始，不是从 1 开始。
PHP版本：4+。
